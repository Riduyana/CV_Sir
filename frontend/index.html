          ## This is the main HTML file for the CV Sir application. It includes a header, a multi-step form for uploading a resume, entering a job title, and pasting a job description. After the analysis is performed, it displays results including summary cards and charts to help users understand how well their resume matches the target role and job description. The file also links to an external CSS stylesheet for styling and a JavaScript file for functionality.
